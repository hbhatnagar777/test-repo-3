Test Automation
================

Test Automation is a python based automation tool for Commvault Software.

Test Automation uses Developer - SDK package to perform operations on a Commcell via WebConsole.

Custom
======

This directory represents the entry point for all the custom test cases designed by end user.
This directory will be preserved for Service Pack update and Release Upgrade.

Note
====
This directory won't be preserved in case of REPAIR or REINSTALL of Service Pack.
